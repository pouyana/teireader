TEIReader Overview
======================================

TEIReader is mutlifunctional tool which has a goal of automation some Linguistic experiments with Text files from `Textgrid <http://textgrid.de/>`_. It uses different machine learning tools and programs. The program is actively maintained in `Github <https://github.com/pouyana/teireader>`_. It is written mainly in python and is compatible with both python versions 2.7/3.x.
