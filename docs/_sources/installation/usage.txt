=======================
Using TEIReader
=======================

Download Latest TEIReader
-------------------------
TEIReader is availbe as a zip package on github.
    - `Download TEIReader <https://github.com/pouyana/teireader/archive/master.zip>`_

Running TEIReader
-----------------------
To run TEIReader execute the following command, after extracting the archive.

    $python teireader.py

You will have several options to choose.
